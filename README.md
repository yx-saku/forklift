# 主な方針
- 衝撃を伴わない異常はいったん無視（人身ヒヤリなど）
- 異常を見逃さないことが優先。絶対平常だと判断できる場合のみ平常にする
  - つまり明らかに平常なものは異常値スコアが極端に低く出すようにしたい


明らかに激しい衝撃音が
  - ある ->異常
  - ない（多少の衝撃音を含む）  
    車体付近に物体が
    - ある  
      ☆作業中に必要な、意図的な衝突である
      - Yes ->平常
      - No ->異常
    - ない  
      ★床に踏んではいけないものが
      - ある ->異常
      - ない ->平常


- ☆の例
  - 積まれたパレットの位置を揃えるため、フォークリフトの爪を（干渉しない位置まで）上げて、車体をぶつける
- ★の例
  - 電源コードなど  
- ★で踏んでいいものの例
  - 工場内の床のつなぎ目のわずかな段差
  - エレベーターや床保護用の鉄板のつなぎ目
  - 側溝のふた
  - アスファルトのひび割れ

入れる特徴量
・音声
・向きの変化量スコア
　・短時間に何度も変わる
　・かつその変化が複数のグリッドで同期している
